# YUR: Void-Full Framework

A "World Computer" prototype that unifies quantum gravity and cosmology through the **Void-Full Framework (VFF)**, powered by \( T = \infty \times 0 \). YUR simulates entropic gravity, quantum entanglement, and dark energy emergence in an interactive, scalable platform. It’s a bold step toward a decentralized, immersive UI (think tesseract desktop) and a "smart world" vision tied to ThingForge.

## 🌌 Why YUR?
- **Unifies Physics**: Models void entropy, cosmological constant \( \Lambda \), and entanglement phase shifts via \( T = \infty \times 0 \).
- **Matches Data**: Predicts \( \Lambda \approx 1.63 \times 10^{-35} \, \text{s}^{-2} \) (aligns with DESI 2026) and entanglement phases (~0.006 rad, testable via MIT oscillators).
- **Open & Extensible**: Built for community forks—add 3D time, neutrino mass (~0.058 eV).
- **Spatial UI**: WebXR frontend visualizes the void-to-cosmos transition, a foundation for spatial computing.
